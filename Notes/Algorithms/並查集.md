# 並查集

## 原理

快速尋找一個點所隸屬的母群體，可以開一個陣列後快速查找母群體的總合或其他狀態。

- 時間複雜度：
  - union : 本身為O(1), 會用到 find 來找點位的根
  - find : O(logn), 優化後 O(1)
- 空間複雜度：
  - O(n) 來開 parents 空間
